This is Binding Component

In this component, will show examples of different data bindings


Interpolation syntax

The Angular uses the {{ templateExpression }} (double curly braces) in the template to denote the interpolation. The syntax is as shown below

<p> Welcome {{ firstName + ' ' + lastName }} </p>

Welcome {{firstName + ' ' +lastName}}

or
<p> Welcome {{ firstName }} {{ lastName }} </p>

Welcome {{firstName}} {{lastName}}

Property Binding

The Property Binding uses the following Syntax

[binding-target]=”binding-source”
Example:
<button class="btn btn-secondary" [disabled]="isDisabled">I am disabled</button>


Event Binding

The Angular event binding consists of two parts

(target-event)="TemplateStatement"
Example:
<button (click)="clickMe()" class="btn btn-secondary">Click Me</button>

You have clicked {{clickCount}}

Two way Data binding

Angular provide a way to achieve the two-way binding using the syntax [()]

Two-way binding syntax:

<someElement [(someProperty)]="value"></someElement>
Example:
<input class="form-control col-sm-4" type="text" name="firstName" [(ngModel)]="firstName">
<input class="form-control col-sm-4" type="text" name="lastName" [(ngModel)]="lastName">
